Идея проекта, функционал.

Python мессенджер, написанный с использованием Qt5, Flask. 

Задействованные библиотеки для:
		Сервера: sqlite3, hashlib, Flask, time
		UI: PyQt5, requests, hashlib, sys.


Описание проекта:
GUI интерфейс написан на Pyqt. При запуске программы появляется окно регистирации/авторизации, после успешного входа в аккаунт 
пользователь попадает в общий чат (Возможны в скором времени личные чаты).
База данных одна - привязана к серверу, если у каждого пользователя будет своя БД - произойдет хаос.
При регистрации отправляется запрос на сервер, где происходит проверка на индивидульность имени, и если такого имени еще нет, то 
пользователь добавляется в базу.
Все пользователи хранятся в БД, и когда при входе в аккаунт мы посылаем запрос на получение списка всех пользователей, сервер 
берет эти данные с БД и отображает на странице /users. Далее, главная программа берет эти данные, которые хранятся как json
объект и выполняет проверку вводимых данных на верность и валидность

Все сообщения также хранятся в БД, прявизанной к серверу

Обязятельный функционал:
	1) Система регистирации/авторизации со свой БД MySQL
	2) Создание ОБЩЕГО чата, для всех пользователей.
	3) Написание сервера, использующего Flask
	4) Хранения паролей в БД в хешированном виде
	5) выход из аккаунта, вход в другой

Желательный фукционал:
	1) Добавление возможности создания личных чатов
	2) Добавление возможности создания групп
	3) Добавление личного кабинета с возможностью редактирования личных данных
	4) Возможность отправки в чат файлов с расширениями (.jpg, .png, .gif)
	5) "Запоминание" устройства, чтобы при каждом запуске программы не входить в аккаунт заново.
